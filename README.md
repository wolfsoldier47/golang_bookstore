**The routes to access the REST API:**
| Action | ROUTE  |METHOD |
| ------- | --- | --- |
| Get_ALL_books | /book/  | GET|
| ADD_A_BOOK	| /book/  | POST|
| GET_BOOK_BY_ID| /book/{bookID} | GET
| UPDATE_BY_ID  | /book/{bookID} | PUT
| DELETE_BY_ID  | /book/{bookID} | DELETE
